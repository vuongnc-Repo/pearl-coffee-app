# Pearl Coffee

A luxury coffee shop management web application built with React + Vite.

## Features

- **Landing Page** - Elegant dark/gold theme with scroll animations
- **Stock Management** - Track inventory with low-stock alerts
- **Employee Workflow** - Kanban task board and shift scheduling
- **Recipes** - Cost calculation with ingredient tracking
- **Orders & Analytics** - Daily order logging with revenue charts

## Tech Stack

- React 18 + Vite
- react-router-dom (HashRouter for static hosting)
- recharts (charts)
- lucide-react (icons)
- framer-motion (animations)
- localStorage (data persistence)
